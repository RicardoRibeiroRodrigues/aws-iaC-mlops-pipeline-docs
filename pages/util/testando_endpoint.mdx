## Testando o deploy do modelo

<Steps>
### Ache o nome dos endpoints:

Esse link vai para a lista dos endpoints, https://us-east-1.console.aws.amazon.com/sagemaker/home?region=us-east-1#/endpoints, apenas modifique para a região que estiver sendo utilizada.

### Veja nos endpoints o nome do endpoint que deseja testar.

![Imagem endpoints](/pipelines_sagemaker.png)

### Pegue o nome do endpoint que deseja testar

Apenas escolher e copiar, para usar no script abaixo.

### Testando o endpoint com o python

```sh
python test_endpoint.py
```

Coloque o nome do endpoint no input do terminal, você deve ver duas saídas em seu console, uma é o dado de teste que está sendo enviado para a API, a segunda é a resposta da API, sendo a variável prevista pelo seu modelo em resposta aos dados enviados.
